# Gestión Universitaria

Sistema de gestión académica desarrollado en Java.

## Descripción
Aplicación web para la administración de una universidad. Permite gestionar:
*   Estudiantes
*   Profesores
*   Cursos
*   Inscripciones
*   Calificaciones

## Tecnologías
*   **Backend**: Java (Javalin)
*   **Frontend**: HTML5, Bootstrap 5, JavaScript
*   **Datos**: JSON (Persistencia en archivo)
*   **Tiempo Real**: WebSockets

## Ejecución
1.  Compilar: `mvn clean package`
2.  Ejecutar: `java -jar target/gestion-universitaria-1.0.jar`
3.  Acceder: `http://localhost:7000` (Admin: `admin` / `admin123`)
