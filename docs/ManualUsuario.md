# Manual de Usuario

## Acceso al Sistema
1. Abrir navegador en `http://localhost:7000`
2. Navegar usando las pestañas superiores.

## Funcionalidades
- **Estudiantes**: Agregar y listar estudiantes.
- **Profesores**: Gestión de docentes.
- **Cursos**: Creación de cursos asignados a profesores.
- **Inscripciones**:
  - Seleccione un estudiante y un curso.
  - Click en "Inscribir".
  - Verá inmediatamente en la tabla inferior las materias que tiene ese estudiante.
- **Reportes**: Vista rápida de conteos totales.

## Persistencia
Los datos se guardan en `data.json`. No borre este archivo si quiere mantener sus datos.
